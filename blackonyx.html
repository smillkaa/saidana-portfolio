Client's Goal: The client requested a single-page website for their nail design studio that eliminates the need for scrolling.

Desktop

Mobile

01/

Approach

To meet the client's requirements, I created a minimalistic website that effectively showcases all the essential information. The website includes details such as the studio's address and prominent buttons for booking appointments and purchasing gift cards.

02/

Process

I took inspiration from the studio's distinctive aesthetic, which features black and white line art covering every aspect, from walls to flooring and furniture. I aimed to reflect this visually in the web design by adhering to a black and white color palette.

03/

Tools

  • HTML + CSS: layout and styling.
  • Photoshop: I used Photoshop to enhance the central image, which is a photograph of the studio's wallpaper.
  • JavaScript: As per the client's request, I incorporated JavaScript functionality that dynamically changes the center image each time the page loads.
